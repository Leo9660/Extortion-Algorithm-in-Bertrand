这里的数据结果来自2019年5月17日在浙江大学社会科学实验室所开展的实验。
result(1-20).csv是每一位实验者以及算法每一轮的决策，low一列是价格较低的一方。
statistic.csv是统计了每一位被试及算法在不同时期（1-500，1-250，251-500）的平均收益，以及该收益对应的合谋程度，并求了总平均数，中位数，方差等统计量。
statistic.csv是统计了每一位被试及算法在不同时期的平均价格，以及该价格对应的合谋程度，并求了总平均数，中位数，方差等统计量。
graph_human.xls是每一位实验者每一轮选择的价格，求出中位数后即可画出趋势图，也就是论文中的价格变化图。
Degree.xls里包含实验者合谋程度（中位数）的变化趋势图。
